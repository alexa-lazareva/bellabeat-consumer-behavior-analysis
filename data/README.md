# Dataset Information

This folder contains cleaned Fitbit activity datasets used for the Bellabeat case study.

# Dataset Source

The original Fitbit Fitness Tracker dataset used in this project is publicly available on Kaggle:

[https://www.kaggle.com/datasets/arashnic/fitbit](https://www.kaggle.com/datasets/arashnic/fitbit)

A cleaned and standardized version of the dataset was used for analysis in SQL, R, and Tableau workflows.
Due to GitHub file-size limitations, the full raw dataset is not stored directly in this repository.
