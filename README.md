# Consumer Behavior Analysis for Bellabeat

## Project Overview

This project analyzes Fitbit smart-device usage data to identify consumer behavior trends and uncover marketing opportunities for Bellabeat, a wellness technology company focused on women’s health.

Using SQL (BigQuery), R, Tableau, and Excel, I explored user activity patterns, engagement behavior, and wellness tracking habits to develop data-driven business recommendations.

---

## Business Objective

The goal of this analysis was to:
- identify behavioral trends in Fitbit user activity data,
- segment users based on activity levels,
- and uncover opportunities to improve user engagement and personalized wellness experiences.

---

## Tools Used

- SQL (BigQuery)
- R (tidyverse, ggplot2)
- Tableau
- Excel

---

## Key Insights

- 35% of users were classified as sedentary
- Manual tracking features showed lower engagement than passive tracking
- Daily steps and calories burned demonstrated a strong positive correlation
- User segmentation revealed significant differences in activity behavior

---

## Strategic Recommendations

Based on the analysis, I recommended:
- progressive fitness goals,
- personalized wellness nudges,
- and lower-friction tracking experiences.

These strategies could help Bellabeat improve user engagement, retention, and product positioning.
