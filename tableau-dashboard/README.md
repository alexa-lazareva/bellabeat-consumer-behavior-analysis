# Tableau Dashboard

To support stakeholder communication and executive reporting, I created an interactive Tableau dashboard focused on user activity segmentation and engagement behavior.

The dashboard highlights:
- user distribution across activity levels,
- engagement patterns among sedentary and active users,
- and behavioral trends related to daily movement and calorie expenditure.

## Key Business Insight

The dashboard reveals that 35% of users fall into the sedentary category, highlighting a strong opportunity for beginner-focused wellness engagement strategies and personalized activity recommendations.
